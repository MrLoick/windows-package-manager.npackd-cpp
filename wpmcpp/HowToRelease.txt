Change the version number in: 
- wpmcpp\WPMUtils.cpp
- wpmcpp\wpmcpp.aip
- wpmcpp\wpmcpp64.aip
- npackdcl\NpackdCL.aip

Translations:
- run lupdate
- update translations
- run lrelease

Tag the version in Mercurial, commit, push.

Execute the following scripts:
- wpmcpp\Build32.bat
- wpmcpp\Build64.bat
- npackdcl\Build32.bat

- wpmcpp\32\Npackd32-X.XX.X.msi
- wpmcpp\32\Npackd32.zip
- wpmcpp\32\wpmcpp_release.map
- wpmcpp\64\Npackd64-X.XX.X.msi
- wpmcpp\64\Npackd64.zip
- wpmcpp\64\wpmcpp_release.map
- npackdcl\32\NpackdCL-X.XX.X.msi
