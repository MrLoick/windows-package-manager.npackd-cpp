Change the version number in wpmcpp\version.txt

Translations:
- run lupdate
- update translations
- run lrelease

Tag the version in Mercurial, commit, push.

Create the following packages from the repository 
(hg clone https://tim.lebedkov@code.google.com/p/windows-package-manager.npackd-cpp/):
- npackd-dev-i686-w64
- npackd-dev-x86_64-w64

Execute the following scripts in npackd-dev-i686-w64:
- wpmcpp\Build32.bat
- npackdcl\Build32.bat
- clu\Build32.bat

Execute the following scripts in npackd-dev-x86_64-w64:
- wpmcpp\Build64.bat
